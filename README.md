# **VNEMOS: Vietnamese Speech Emotion Speech Dataset**

This repository of **VNEMOS**, a curated Vietnamese speech emotion dataset created for research on Speech Emotion Recognition (SER), affective computing, and cognitive behavior analysis.

## Overview

VNEMOS contains **250 emotional audio clips (~30 minutes total)** extracted from Vietnamese movies, TV series, and live shows.
The dataset is **balanced**, with **50 segments** for each of the following five emotions:
